# float元素的五大特性

  1. 会以某种方式将浮动元素从文档的正常流中删除，采用css的特有方式，‘自成一派’
  2. 其他内容会”环绕“浮动元素
  3. 浮动元素周围的外边距不会合并；（涉及BFC）
  4. 如果确实要浮动一个非替换元素，则必须为该元素声明一个width，否则，根据css规范，元素宽度趋近于零，如果中间有文本，则最小为一个单词文本宽度
  5. 浮动元素的包含块是其最近的块级祖先元素
  6. 浮动元素会生成一个块级框
