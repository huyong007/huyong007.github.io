    1. 取整：不要用parseInt，多用Math.trunc()
    • 一是因为parseInt你需要传第二个值，是使用什么进制，不通过浏览器默认值不同，大部分是10进制，你倒霉起来都找不到bug
    parseInt(0.0000003) // 3
    1. 带小数点的比较
    • 涉及到小数点后面位数的比较最好是取到一定的位数，比如都是取小数点后两位，然后*100再来比较，这是比较安全的比较方式，银行js业务处理时可信用法
    1. 假设法在数组类算法题中的运用
    • 求一个数组中的最大值，假设maxValueIndex=0是最大值，然后for循环和假设值比较，再来更新maxValueIndex
    // 求数组arr = [1,2,3,4,9,7]中的最大值
let maxIndex = 0;
for(let i=0;i<arr.length;i++){
    arr[i]>arr[maxIndex]?maxIndex=i:'';
}
